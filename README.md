#Face Score System

##How to build
- please refer to [face-analysis-sdk](https://github.com/ci2cv/face-analysis-sdk) for more details
```shell
cd build
cmake ..
make
```
after that, just copy some images from computer to build directory and 
```shell
face-fit test.jpg
```
and enjoy!
